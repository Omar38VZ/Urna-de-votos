## Programa de porcentajes de votos
Este programa sencillo echo en Java permite calcular el porcentaje de votos obtenidos por cada candidato en una urna. El sistema solicita el número del candidato cada vez que se extrae un voto y finaliza cuando el usuario ingresa el valor 0. Al terminar, muestra el total de votos y el porcentaje correspondiente para cada candidato.

La implementación se realizó utilizando arreglos como cubetas de almacenamiento, donde cada posición del arreglo representa a un candidato y acumula la cantidad de votos recibidos. Esta estructura la elegí por su simplicidad, eficiencia.
